# Arkm热钱包爬虫 @ KrsMt.

1. 填写配置文件
  - 1. `args.txt`
       
       填写需要查询的实体信息，大名在前，小名在后，用英文半角逗号隔开。例如：
       
       ```
       OKX,okx
       Binance,binance
       ```
  - 2. `config.txt`
       
       填写查询参数。第一行`num=`代表单次查询数量，第二行`offset=`代表查询次数。其中，单次查询数量越大，服务器响应越慢。

2. 运行`main.py`，等待即可。程序运行时会打印日志，以便获得状态。同时，在主程序同目录下创建名为`result`的文件夹，用于存储运行结果。运行结果以`.csv`格式保存。
